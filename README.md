
## She Can Foundation Website
A modern, visually engaging, and fully responsive NGO website designed to inspire action and raise awareness around women empowerment.

The **She Can Foundation** website was created as a concept project for organizations working towards education, healthcare awareness, skill development, and community support for women and girls. The design focuses on storytelling, accessibility, and emotional engagement while maintaining a clean and professional user experience.


## Project Overview

She Can Foundation is a youth-driven initiative that aims to create opportunities, confidence, and independence for women through education, mentorship, healthcare awareness, and skill-building programs.

This website serves as a digital platform to showcase the organization's mission, highlight its impact, present ongoing programs, and encourage visitors to participate through volunteering, donations, or community engagement.

The project combines modern design principles with smooth interactions to create a memorable browsing experience.


## Features

### Modern Hero Section

* Full-screen immersive landing section
* Strong visual storytelling
* Clear call-to-action buttons
* NGO-focused branding and messaging

### Responsive Design

* Optimized for desktop, tablet, and mobile devices
* Flexible layouts using CSS Grid and modern responsive techniques

### Dark & Light Mode

* One-click theme switching
* Smooth transition effects
* Improved accessibility and user preference support

### Impact Statistics

* Dedicated section to showcase organization achievements
* Easily customizable metrics and counters

### Program Showcase

* Beautiful card-based layout
* Highlight educational, mentorship, and awareness initiatives
* Interactive hover animations

### Gallery Section

* Responsive masonry-style gallery
* Visual representation of community activities and events

### Interactive Experience

* Custom animated cursor
* Scroll progress indicator
* Reveal-on-scroll animations
* Smooth navigation and transitions

### Call To Action

* Volunteer and donation engagement section
* Encourages community participation and support


## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Google Fonts
* Intersection Observer API
* CSS Grid & Flexbox

No external frameworks or libraries were used, making the project lightweight and easy to customize.



## Design Highlights

The website follows a vibrant visual identity built around:

* Pink accents representing empowerment and confidence
* Purple gradients representing growth and transformation
* Yellow highlights symbolizing optimism and opportunity
* Modern typography combinations for a bold and inspiring appearance

The overall design balances professionalism with warmth, helping visitors connect emotionally with the mission.
